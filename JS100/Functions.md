## Cite the Author

```javascript 
function cite(author, quote){
	console.log(author + " said: " + '"' + quote + '"');
}
```

Gotcha on quotes. 

Better solution 

```javascript 
function cite(author, quote) {
  console.log(`${author} said: "${quote}"`);
}
```
