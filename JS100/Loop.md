## Loop and Log

```javascript
for (let i = 0; i <= 10; i += 2) {
  console.log(i) 
}
// 0, 2, 4, 6, 8 ,10
```

## Countdown
```javascript
for (let i = 10; i >= 1; i--) {
  console.log(i);
}
console.log("Launch!");
```
## Triple Greeting
```
var greeting = 'Aloha';
var i = 3; 
while(i > 0){
  console.log(greeting);
  i--;
}
```
